Searchable Map Template

Turn a spreadsheet in to a searchable map

You want to put your data on a map. This is a free, open source tool to help you do it.

Features:

  • clean, full screen layout
  • address search (with variable radius)
  • geolocation (find me!)
  • results count (using Google's visualization API)
  • ability to easily add additional search filters (checkboxes, sliders, etc)
  • all done with HTML, CSS and Javascript - no server side code required

Technologies used:

  • jQuery
  • Google Maps API V3
  • Google Fusion Tables
